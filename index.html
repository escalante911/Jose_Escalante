Tema: Selectores CSS

Los selectores son patrones que indican a qué elementos del documento se aplican las reglas de estilo. A continuación, un resumen de los más usados con ejemplos breves.

1. Básicos

  • Universal *: selecciona todos los elementos.
  • Tipo p, h1: por nombre de etiqueta.
  • Clase .btn: por clase.
  • ID #hero: por identificador (único en la página).
/* Ejemplos */
* { box-sizing: border-box; }
p { line-height: 1.6; }
.btn { padding: .5rem 1rem; }
#hero { min-height: 50vh; }

2. De atributo

  • a[target]: con atributo target.
  • input[type="email"]: atributo con valor exacto.
  • [data-role^="btn-"]: valor que empieza con texto.
  • [aria-label*="cerrar"]: valor que contiene texto.
a[target] { text-decoration: underline; }
input[type="email"] { border: 1px solid #334155; }
[data-role^="btn-"] { cursor: pointer; }

3. Combinadores

  • Descendiente ul li: cualquier nivel dentro.
  • Hijo directo ul > li: un nivel abajo.
  • Hermano adyacente h2 + p: el primer hermano siguiente.
  • Hermano general h2 ~ p: todos los hermanos siguientes.
ul > li { list-style: disc; }
h2 + p { margin-top: .25rem; }

4. Pseudoclases y pseudoelementos

  • Pseudoclases: estado o posición (:hover, :focus, :first-child, :nth-child(2n)).
  • Pseudoelementos: parte virtual del elemento (::before, ::after, ::selection).
a:hover { opacity: .85; }
.card::before { content: ""; display:block; }

5. Especificidad (resumen rápido)

La prioridad de aplicación de estilos (de menor a mayor) suele considerar: selector de tipo < clase < ID < estilos en línea. Evita usar !important salvo casos necesarios.
